# ✈️ MLO Conteneur d'Aéroport - FiveM

## Description
Ce MLO (Map Location Object) ajoute un conteneur interactif à l'aéroport, entièrement accessible de l'intérieur. Il est conçu pour s'intégrer de manière réaliste à l'environnement existant de GTA V et utilise des **props déjà présents dans le jeu** pour garantir la compatibilité et les performances.  

---

## 🛠️ Contenu
- Conteneur intérieur fonctionnel
- Props GTA V intégrés pour meubler l'intérieur
- Points d'entrée et de sortie configurés
- Compatible avec la plupart des serveurs FiveM

---

## ⚙️ Installation
1. Télécharger ou cloner le dossier du MLO.
2. Copier le dossier dans votre répertoire `resources` du serveur FiveM.
